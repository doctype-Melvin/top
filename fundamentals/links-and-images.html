The following contains info about the four different image formats used on the web.


The JPG


This is a JPG image. JPGs are goot for photos. They don't allow for transparent pixels though.


The GIF


This is a GIF image - the go-to option for simple animations. They're limited in terms of color palette - so better not use 'em for photos.


The PNG


This is a PNG image. Use PNG for anything but photos because a PNG would be way bigger than a JPG.

Icons, technical diagrams, logos etc. should be used as PNGs.


The SVG


This is a SVG which is a vector-based graphic. It can scale up or down to any dimension without loss of quality.

Responsive design is the home of the SVG.
